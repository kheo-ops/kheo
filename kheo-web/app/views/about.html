Kheo is an agentless application dedicated to servers management, including softwares inventory, os and network informations. Kheo relies on SSH to communicate with the servers to manage so it does not need any specific configuration. As a constraint, it needs a key to contact remote hosts. To register servers, you have many solutions:
  • Using an inventory file just like Ansible does.
  • Using the API
  • Using the webapp
Once your servers have been registered, you can obtain informations like:
  • Network interfaces and IP
  • OS type and version
  • Users
  • Resources (ram, disk, cpu, ...)
  • Running processes
  • Installed packages
Kheo is build with the following technologies:

MongoDB

MongoDB is an open-source document database, and the leading NoSQL database.

AngularJS

MVW framework to build client side single page applications.

Dropwizard

Dropwizard pulls together stable, mature libraries from the Java ecosystem into a simple, light-weight package that lets you focus on getting things done.
